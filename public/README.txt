Lecture's Digest - Static HTML/JS/CSS Version

Pages:
- index.html: Landing page
- upload.html: Upload and summarize .txt files
- input.html: Manual text input and summarization

APIs:
- Google Gemini API for summarization
- YouTube Data API for video recommendations

How to use:
- Open index.html in your browser
- Use Upload or Input pages for summarization
- All logic runs in the browser (no backend required)
